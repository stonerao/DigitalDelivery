## 更新项目

**接口地址**:`/api/handover/projects/update`

**请求方式**:`POST`

**请求数据类型**:`application/x-www-form-urlencoded,application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
{
  "id": "",
  "projectName": "",
  "projectInfo": ""
}
```

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|projectUpdateRequest|更新项目请求|body|true|ProjectUpdateRequest|ProjectUpdateRequest|
|&emsp;&emsp;id|项目ID||true|string||
|&emsp;&emsp;projectName|项目名称||false|string||
|&emsp;&emsp;projectInfo|项目信息（JSON字符串）||false|string||

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultProjectResponse|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||ProjectResponse|ProjectResponse|
|&emsp;&emsp;id|项目ID|string||
|&emsp;&emsp;projectName|项目名称|string||
|&emsp;&emsp;projectInfo|项目信息（JSON字符串）|string||
|&emsp;&emsp;createdAt|创建时间|string(date-time)||
|&emsp;&emsp;updatedAt|更新时间|string(date-time)||
|&emsp;&emsp;createdBy|创建人|string||
|&emsp;&emsp;updatedBy|更新人|string||

**响应示例**:

```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": "",
		"projectName": "",
		"projectInfo": "",
		"createdAt": "",
		"updatedAt": "",
		"createdBy": "",
		"updatedBy": ""
	}
}
```
