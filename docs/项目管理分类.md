

## 重命名项目类型


**接口地址**:`/api/handover/projectTypes/rename`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": "",
  "name": ""
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|projectTypeRenameRequest|项目类型重命名请求|body|true|ProjectTypeRenameRequest|ProjectTypeRenameRequest|
|&emsp;&emsp;id|类型ID||true|string||
|&emsp;&emsp;name|新名称||true|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultProjectTypeTreeNode|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||ProjectTypeTreeNode|ProjectTypeTreeNode|
|&emsp;&emsp;id|类型ID|string||
|&emsp;&emsp;name|类型名称|string||
|&emsp;&emsp;parentId|父类型ID|string||
|&emsp;&emsp;sortOrder|排序|integer(int32)||
|&emsp;&emsp;children|子类型列表|array|ProjectTypeTreeNode|


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": "",
		"name": "",
		"parentId": "",
		"sortOrder": 0,
		"children": [
			{
				"id": "",
				"name": "",
				"parentId": "",
				"sortOrder": 0,
				"children": [
					{}
				]
			}
		]
	}
}
```
