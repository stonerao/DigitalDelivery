

## 更新关联关系


**接口地址**:`/api/search-nav/relationRecords/update`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": "",
  "type": "",
  "sourceKind": "",
  "sourceId": "",
  "targetKind": "",
  "targetId": ""
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |

|&emsp;&emsp;type|关联类型||false|string||

|&emsp;&emsp;targetId|目标ID||false|string||

| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultRelationRecordResponse|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||RelationRecordResponse|RelationRecordResponse|
|&emsp;&emsp;id|关联记录ID|string||
|&emsp;&emsp;type|关联类型|string||
|&emsp;&emsp;sourceKind|来源类型|string||
|&emsp;&emsp;sourceId|来源ID|string||

|&emsp;&emsp;updatedAt|更新时间|string(date-time)||
|&emsp;&emsp;createdBy|创建人|string||

**响应示例**:
