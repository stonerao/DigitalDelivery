## 快速上传（文档）

**接口地址**:`/api/handover/dashboard/quickUpload`

**请求方式**:`POST`

**请求数据类型**:`application/x-www-form-urlencoded,application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|strings|string|body|true|array||
|dashboardQuickUploadRequest|DashboardQuickUploadRequest|body|true|DashboardQuickUploadRequest|DashboardQuickUploadRequest|
|&emsp;&emsp;transferDepth|||false|string||
|&emsp;&emsp;projectId|||false|string||

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultDashboardQuickUploadResponse|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||DashboardQuickUploadResponse|DashboardQuickUploadResponse|
|&emsp;&emsp;uploadedCount||integer(int32)||
|&emsp;&emsp;documentIds||array|string|

**响应示例**:

```javascript
{

	"message": "",
	"data": {
		"uploadedCount": 0,
		"documentIds": []
	}
}
```
