## 单条校验

**接口地址**:`/api/handover/kks/validateOne`

**请求方式**:`POST`

**请求数据类型**:`application/x-www-form-urlencoded,application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
{
  "kks": ""
}
```

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|kksValidateRequest|KKS校验请求|body|true|KksValidateRequest|KksValidateRequest|
|&emsp;&emsp;kks|KKS编码||true|string||

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultKksValidateTaskResponse|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||KksValidateTaskResponse|KksValidateTaskResponse|
|&emsp;&emsp;taskId||string||
|&emsp;&emsp;status||string||

**响应示例**:

```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"taskId": "",
		"status": ""
	}
}
```

## 导入台账数据

**接口地址**:`/api/handover/kks/import`

**请求方式**:`POST`

**请求数据类型**:`multipart/form-data`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
const formData = new FormData();
files.forEach(file => {
  formData.append("files", file);
});
formData.append("importMode", "merge");
formData.append("note", "测试导入数据");
```

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|files|导入文件，支持多个|formData|true|file array||
|importMode|导入模式(merge/replace)|formData|false|string||
|note|导入说明|formData|false|string||

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultKksImportTaskResponse|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||KksImportTaskResponse|KksImportTaskResponse|
|&emsp;&emsp;taskId||string||
|&emsp;&emsp;version||KksVersionResponse|KksVersionResponse|
|&emsp;&emsp;&emsp;&emsp;id||string||
|&emsp;&emsp;&emsp;&emsp;name||string||
|&emsp;&emsp;&emsp;&emsp;time||string||
|&emsp;&emsp;&emsp;&emsp;note||string||

**响应示例**:

```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"taskId": "",
		"version": {
			"id": "",
			"name": "",
			"time": "",
			"note": ""
		}
	}
}
```
