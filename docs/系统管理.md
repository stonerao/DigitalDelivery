## 重命名系统节点

**接口地址**:`/api/handover/systemNodes/rename`

**请求方式**:`POST`

**请求数据类型**:`application/x-www-form-urlencoded,application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
{
  "id": "",
  "name": ""
}
```

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|systemNodeRenameRequest|重命名系统节点请求|body|true|SystemNodeRenameRequest|SystemNodeRenameRequest|
|&emsp;&emsp;id|节点ID||true|string||
|&emsp;&emsp;name|新名称||true|string||

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultSystemNodeTreeNode|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||SystemNodeTreeNode|SystemNodeTreeNode|
|&emsp;&emsp;name|节点名称|string||
|&emsp;&emsp;code|节点编码|string||
|&emsp;&emsp;type|节点类型|string||
|&emsp;&emsp;description|描述|string||
|&emsp;&emsp;children|子节点列表|array|SystemNodeTreeNode|

**响应示例**:

```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"name": "",
		"code": "",
		"type": "",
		"description": "",
		"children": [
			{
				"name": "",

				"children": [
					{}

```
