

## 快速上传（文档）


**接口地址**:`/api/handover/dashboard/quickUpload`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|strings|string|body|true|array||
|dashboardQuickUploadRequest|DashboardQuickUploadRequest|body|true|DashboardQuickUploadRequest|DashboardQuickUploadRequest|
|&emsp;&emsp;transferDepth|||false|string||
|&emsp;&emsp;projectId|||false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultDashboardQuickUploadResponse|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||DashboardQuickUploadResponse|DashboardQuickUploadResponse|
|&emsp;&emsp;uploadedCount||integer(int32)||
|&emsp;&emsp;documentIds||array|string|


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"uploadedCount": 0,
		"documentIds": []
	}
}
```

## 仪表盘统计


**接口地址**:`/api/handover/dashboard/stats`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|projectId|项目ID|query|false|string||
|dateFrom|统计开始日期|query|false|string||
|dateTo|统计结束日期|query|false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultDashboardStatsResponse|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||DashboardStatsResponse|DashboardStatsResponse|
|&emsp;&emsp;documentCount|文档总数|integer(int64)||
|&emsp;&emsp;modelCount|模型总数|integer(int64)||
|&emsp;&emsp;kksCount|KKS总数|integer(int64)||
|&emsp;&emsp;activeProjectCount|活跃项目数|integer(int64)||
|&emsp;&emsp;documentChangeRate|文档变化率|number(double)||
|&emsp;&emsp;modelChangeRate|模型变化率|number(double)||
|&emsp;&emsp;kksChangeRate|KKS变化率|number(double)||
|&emsp;&emsp;activeProjectChangeRate|活跃项目变化率|number(double)||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"documentCount": 0,
		"modelCount": 0,
		"kksCount": 0,
		"activeProjectCount": 0,
		"documentChangeRate": 0,
		"modelChangeRate": 0,
		"kksChangeRate": 0,
		"activeProjectChangeRate": 0
	}
}
```
