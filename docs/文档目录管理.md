

## 重命名目录


**接口地址**:`/api/handover/docFolders/rename`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": "",
  "name": ""
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|docFolderRenameRequest|重命名目录请求|body|true|DocFolderRenameRequest|DocFolderRenameRequest|
|&emsp;&emsp;id|目录ID||true|string||
|&emsp;&emsp;name|新名称||true|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultDocFolderTreeNode|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||DocFolderTreeNode|DocFolderTreeNode|
|&emsp;&emsp;id||string||
|&emsp;&emsp;name||string||
|&emsp;&emsp;parentId||string||
|&emsp;&emsp;children||array|DocFolderTreeNode|
|&emsp;&emsp;isDefault||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": "",
		"name": "",
		"parentId": "",
		"children": [
			{
				"id": "",
				"name": "",
				"parentId": "",
				"children": [
					{}
				],
				"isDefault": true
			}
		],
		"isDefault": true
	}
}
```
