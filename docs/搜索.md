## 全局搜索导航

**接口地址**:`/api/search-nav/search`

**请求方式**:`POST`

**请求数据类型**:`application/x-www-form-urlencoded,application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
{
  "query": "10LAC10AP001",
  "scopeKeys": [
    "documents",
    "models",
    "devices"
  ]
}
```

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|searchNavRequest|搜索导航请求|body|true|SearchNavRequest|SearchNavRequest|
|&emsp;&emsp;query|搜索关键词||true|string||
|&emsp;&emsp;scopeKeys|搜索范围（documents/models/devices）||false|array|string|

**响应状态**:

| 状态码 | 说明 | schema |
| -------- | -------- | ----- |
|200|OK|ResultListSearchResultItem|

**响应参数**:

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|code||integer(int32)|integer(int32)|
|message||string||
|data||array|SearchResultItem|
|&emsp;&emsp;id|ID|string||
|&emsp;&emsp;kind|类型标识（doc/model/kks）|string||
|&emsp;&emsp;scope|作用域（documents/models/devices）|string||
|&emsp;&emsp;title|标题|string||
|&emsp;&emsp;type|类型描述|string||
|&emsp;&emsp;kks|KKS编码|string||
|&emsp;&emsp;description|描述信息|string||

**响应示例**:

```javascript
{
	"code": 0,
	"message": "",
	"data": [
		{
			"id": "doc-2",
			"kind": "doc",
			"scope": "documents",
			"title": "设备数据表.xlsx",
			"type": "文档",
			"kks": "10LAC10AP001",
			"description": "关联系统：给水泵系统（10LAC10）"
		}
	]
}
```
