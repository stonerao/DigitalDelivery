# 三维模型查看器（viewer3d）测试用例

范围：
- 页面：[src/views/visualization/viewer3d/index.vue](../../src/views/visualization/viewer3d/index.vue)
- 全屏页：[src/views/visualization/viewer3d/fullscreen.vue](../../src/views/visualization/viewer3d/fullscreen.vue)
- 组件：[src/3d/components/ThreeModelViewer.vue](../../src/3d/components/ThreeModelViewer.vue)
- 加载器：[src/3d/loaders/loadModel.js](../../src/3d/loaders/loadModel.js)

前置条件：
- 已安装依赖：`pnpm i`
- 建议 Node 版本满足 `20.19+` 或 `22.12+`（Vite 要求）
- 启动：`pnpm dev`，浏览器访问“可视化 → 三维查看”或直接访问 `/visualization/3d-viewer`

测试数据建议：
- 基础模型：`public/model/bim-test.glb`（如果存在）
- Draco 压缩模型：任意 draco 压缩过的 `.glb/.gltf`
- IFC 样例：`.ifc`
- OBJ 样例：`.obj`（如有 `.mtl` 可一并测试）

---

## A. 流程测试（用户操作流程）

### A-1 模型加载流程

**TC-A1-01 上传 GLB 模型**
- 步骤：点击“上传模型” → 选择 `.glb`
- 期望：模型进入渲染区；左上角状态由“加载中”变为“就绪”；面数/统计可显示

**TC-A1-02 上传 GLTF 模型**
- 步骤：上传 `.gltf`（如依赖外部 bin/贴图，确保路径可访问）
- 期望：同上；若资源缺失应出现明确错误提示

**TC-A1-03 上传 IFC 模型**
- 步骤：上传 `.ifc`
- 期望：能加载并显示；无报错；可交互

**TC-A1-04 上传 OBJ 模型**
- 步骤：上传 `.obj`
- 期望：能加载并显示；若无材质也能以默认材质显示

**TC-A1-05 URL 加载（有效 URL）**
- 步骤：在“输入模型 URL”填入可访问的 glb/gltf/ifc/obj 地址 → 点击“加载URL”
- 期望：模型加入列表并加载；加载进度正常

**TC-A1-06 URL 加载（无效 URL）**
- 步骤：输入不存在的地址或 404 地址 → 加载
- 期望：出现错误提示；页面不崩溃；可继续加载其它模型

**TC-A1-07 下拉选择已加载模型**
- 步骤：先加载多个模型 → 下拉切换
- 期望：切换后渲染区更新为新模型；工具状态保持可用

**TC-A1-08 URL 参数加载（联动入口）**
- 步骤：直接访问 `/visualization/3d-viewer?url=...&name=...&id=...`
- 期望：自动加载对应模型；标题/名称显示合理；下拉列表出现该项

### A-2 工具切换流程

**TC-A2-01 旋转/平移/缩放切换**
- 步骤：依次切换“旋转/平移/缩放” → 鼠标拖拽、滚轮
- 期望：交互行为符合模式；切换提示文案正确

**TC-A2-02 测量模式**
- 步骤：切换“测量” → 依次点击两点
- 期望：显示测量线/距离；“清除测量”可清空

**TC-A2-03 拾取模式**
- 步骤：切换“拾取” → 悬停/点击构件
- 期望：悬停高亮、点击选中；右侧抽屉显示属性；关闭抽屉后取消选中

### A-3 视角操作流程

**TC-A3-01 预设视角切换**
- 步骤：点击“等轴测/顶视图/前/后/左/右/底”
- 期望：相机视角切换；提示文案正确

**TC-A3-02 视图重置**
- 步骤：拖动改变视角 → 点击“重置”
- 期望：回到适配视野的默认视角；提示“视图已重置”

**TC-A3-03 缩放按钮**
- 步骤：点击“放大/缩小”
- 期望：相机距离改变；不会穿模/远离到看不见

**TC-A3-04 书签保存与应用**
- 步骤：调整视角 → 点击“书签”保存 → 点击书签项应用
- 期望：能恢复保存时视角；提示文案正确

### A-4 全屏查看流程

**TC-A4-01 进入全屏**
- 步骤：在主视图区域点击（触发 `scene-click`）
- 期望：进入 `/visualization/3d-fullscreen?url=...&name=...`；加载同一模型

**TC-A4-02 从全屏返回**
- 步骤：全屏页点击“返回”
- 期望：返回上一页；若无历史则回到 `/visualization/3d-viewer` 并保留 query

### A-5 视频联动流程

**TC-A5-01 选择区域并跳转**
- 步骤：选择区域（锅炉/控制室/冷却塔/全部）→ 点击跳转视频联动
- 期望：进入 `/visualization/video?region=...`；视频页显示对应联动信息

## B. 功能测试（具体功能点）

### B-1 模型加载功能

**TC-B1-01 加载进度**
- 步骤：加载大模型（或限速网络）
- 期望：显示进度；进度回调不会出现 NaN；完成后变“就绪”

**TC-B1-02 错误提示**
- 步骤：加载不支持格式 / 404 / 跨域失败
- 期望：错误提示可见且可理解；不影响继续操作

**TC-B1-03 Draco 解码**
- 步骤：加载 draco 压缩 glb/gltf
- 期望：成功加载；Network 可见访问 `/draco/` 解码文件

### B-2 交互控制

**TC-B2-01 控制器阻尼**
- 步骤：旋转拖拽后松开
- 期望：有轻微阻尼衰减（如设计如此），不卡顿

**TC-B2-02 拾取过滤**
- 步骤：拾取模式点击网格/坐标轴/测量对象
- 期望：不应选中 helper/测量对象

### B-3 视图控制

**TC-B3-01 透明度切换**
- 步骤：点击“透明”按钮
- 期望：材质透明度变化；再次点击恢复

**TC-B3-02 性能统计**
- 步骤：勾选“性能统计”
- 期望：显示统计面板；关闭后隐藏

**TC-B3-03 剖切平面**
- 步骤：开启“剖切”→ 启用 X/Y/Z 勾选 → 拖动滑块
- 期望：剖切平面即时生效；关闭剖切后恢复

### B-4 相机功能

**TC-B4-01 透视/正交切换**
- 步骤：点击“投影”
- 期望：在透视/正交间切换；左上角有“正交”标记

### B-5 截图与书签

**TC-B5-01 截图下载**
- 步骤：点击“截图”
- 期望：下载 png；图片内容与当前视图一致

### B-6 构件选择

**TC-B6-01 抽屉开关**
- 步骤：拾取构件 → 抽屉出现 → 点击关闭
- 期望：抽屉关闭；选中状态清除

## C. 边界情况

**TC-C-01 空模型**
- 步骤：选择“（未加载）”或清空 URL
- 期望：显示空态；无报错

**TC-C-02 不支持的文件格式**
- 步骤：上传 `.txt` 等
- 期望：给出“仅支持 …”提示且不加载

**TC-C-03 URL 带 query/hash**
- 步骤：加载 `xxx.glb?token=...#v=1`
- 期望：格式识别正常，能加载
